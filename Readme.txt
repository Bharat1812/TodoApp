Steps to run the application

For React :

Step 1 - Open terminal and run the command "npm install", It will install all the modules I am using in the application

Step 2 - After installing node modules Run the command "npm start"

React Application is started

For Node :

Step 1 - Open terminal and run the command "npm install", It will install all the modules I am using in the application

Step 2 - After installing node modules Run the command "npm start"

Node Application is started
